# Secret Santa

A simple but effective web application for coordinating Secret Santa gift exchanges. Add participants, set gift preferences and exclusions, then let the app handle random matching while ensuring no one gets themselves or excluded pairings. Email notifications keep the surprise intact while making sure everyone knows their assignment.

## Features

- Automated random matching algorithm
- Exclusion rules for couples/families
- Gift preference tracking
- Email notification system
- Mobile-friendly interface

## Tech Stack

React, Node.js

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Running

```bash
npm start
```
